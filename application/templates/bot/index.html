Tornium Bot

The Tornium bot is a Discord Bot implemented from scratch with the Discord API for Torn City.

For more information, contact tiksan on Discord. If you want to host this bot on your own server, please follow these the steps listed in the documentation.

Features

  • Verification - a verification utility to give user's their Torn name and ID along with faction role and/or position roles
  • Banking - a faction utility for requesting money from the faction vault
  • Retaliation - a utility to provide notifications of retaliations
  • OC Notifications - a utility to provide notifications when organized crimes are ready and/or delayed
  • [BETA] Stakeouts - a slash command-based utility to watch factions and users for changes in specified triggers
