Tornium Notifications

Tornium notifications is a customizable notification engine built with Lua to provide users with custom notifications in Torn (in-development) and in Discord servers. For information on setting up notifications in your Discord servers, see the documentation.

Features

  • Lua Bindings - Notification triggers are written in Lua
  • Liquid Message Templates - Message templates for Discord messages built with Liquid
  • Pre-built Notifications - There are several pre-built notification triggers that you can quickly use in your servers
