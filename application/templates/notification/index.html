Tornium Notifications

Tornium notifications is a customizable notification engine built with Lua to provide users with custom notifications in Torn (in-development) and in Discord servers.

For more information, contact tiksan on Discord. If you want to host this bot on your own server, please follow these the steps listed in the documentation.

Features

  • Lua Bindings - Notification triggers are written in Lua
  • Liquid Message Templates - Message templates for Discord messages built with Liquid

Supported Resources

  • User - In-Development
  • Faction - In-Development
  • Company - In-Development
