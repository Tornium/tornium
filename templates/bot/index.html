Tornium Bot

The Tornium bot is a Discord Bot implemented from scratch with the Discord API for Torn City.

For more information, contact tiksan on Discord. If you want to host this bot on your own server, please follow these the steps listed here.

Features

  • Verification - a verification utility to give user's their Torn name and ID along with faction role and/or position roles
  • Banking - a faction utility for requesting money from the faction vault
  • Stakeouts - a utility to watch factions, users, and companies for changes in specified triggers

Source Code

The officially hosted Tornium Discord bot is closed source due to the inclusion of paid features, however the original open-source version can be found on GitHub. The original open-source version of Tornium has stopped being maintained though due to lack of availability on the developer's part.
